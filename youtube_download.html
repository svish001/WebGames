📺 YouTube Downloader

⚠️
Requires local server — Downloading videos is a server-side operation that cannot run in a browser. You must run hub_server.py on your computer, then use this tool locally.
How to use locally
1
Install yt-dlp: pip install yt-dlp
2
Run the server: python hub_server.py
3
Open http://127.0.0.1:8765 in your browser
4
Click the YouTube Downloader tile and paste your URL
